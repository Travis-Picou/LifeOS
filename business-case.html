Value drivers & willingness to pay (EVC)

Each driver = Quantity × Unit value × Attribution %. Differentiation value sits on top of the next-best-alternative (reference value) to set the EVC ceiling. Price = reference + capture% × differentiation.

Differentiation value
Capture 20–50% of differentiation value (default 30%); leave the rest as the customer's surplus so the deal is obviously worth it. Price must sit between reference value (floor) and EVC (ceiling).
